# NVIDIA Fundamentals of Deep Learning

This repository contains my coursework and projects from the **NVIDIA Fundamentals of Deep Learning** course. The course covers essential deep learning concepts and practical implementations using modern frameworks.

## Course Content

### Jupyter Notebooks

1. **00_jupyterlab.ipynb** - Introduction to JupyterLab environment
2. **01_mnist.ipynb** - MNIST digit classification (basic neural networks)
3. **02_asl.ipynb** - American Sign Language recognition
4. **03_asl_cnn.ipynb** - ASL recognition using Convolutional Neural Networks
5. **04a_asl_augmentation.ipynb** - Data augmentation techniques for ASL
6. **04b_asl_predictions.ipynb** - Making predictions on ASL data
7. **05a_doggy_door.ipynb** - Dog classification project
8. **05b_presidential_doggy_door.ipynb** - Presidential dog Bo recognition
9. **06_nlp.ipynb** - Natural Language Processing fundamentals
10. **07_assessment.ipynb** - Course assessment and final project

### Datasets

The `data/` folder contains various datasets used throughout the course:
- **MNIST** - Handwritten digit dataset
- **ASL Data** - American Sign Language images
- **Doggy Door Data** - Dog classification images
- **Presidential Doggy Door** - Bo (presidential dog) dataset
- **Fruits** - Fresh and rotten fruit classification
- **NYT Dataset** - New York Times articles for NLP tasks

### Utilities

- **utils.py** - Helper functions and utilities used across notebooks
